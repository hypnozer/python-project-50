# Difference Calculator

Difference Calculator is a CLI utility that compares two configuration files and shows the difference between them.

The utility supports:
- JSON and YAML input files
- stylish output format
- plain output format
- json output format

- Python 3.10+
- [uv](https://github.com/astral-sh/uv)
